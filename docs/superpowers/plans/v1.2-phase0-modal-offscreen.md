# Plan: v1.2 Phase 0 — Modal off-screen fix (#8)

**Spec:** `docs/superpowers/specs/2026-06-18-phase0-modal-offscreen.md`
**Branch:** `fix/v1.2-phase0-modal-offscreen` (off `origin/main`)

## Scope / Goal

Cap `DialogContent` at the viewport height with internal scroll so tall dialogs (PinSheet, PrinterSheet, mgr admin dialogs) stop clipping their header/footer off-screen on the booth tablet. One className edit on the shared primitive fixes all 11 dialog instances.

## File Changes

1. **`src/components/ui/dialog.tsx`** (modify) — add `max-h-[calc(100dvh-2rem)] overflow-y-auto` to the `DialogContent` className.
2. **`src/components/ui/__tests__/dialog.test.tsx`** (create) — one class-presence guard test (deletion-guard, per spec).
3. **`docs/CHANGELOG.md`** (modify) — Phase 0 entry.
